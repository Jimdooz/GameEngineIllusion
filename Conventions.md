# Conventions nommages

## Brief

Le présent fichier présente les différentes conventions qui seront utilisées dans le présent projet à des fins de facilité d'écriture et d'inter-compréhension.

## Convention de nommage

- Macro/constexpr : `MA_SUPER_MACRO`
- Classe :   Camel Case avec majuscule première lettre (MaSuperClasse)
- Attribut : `m_uMonAttribut` (Unsigned Int Member of a classe)
- Exemples de types: u (Unsigned Int) i (Int), b (Boolean), p (Pointer), f (Float), d(Double), o(Object), a(Array), v(Vector), ...
- Méthode Public : Camel Case (MaSuperMethode)
- Méthode Protected/Private : Camel Case avec minuscule première lettre(maSuperAutreMethode)
- Namespace : monamespace
